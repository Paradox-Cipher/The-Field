# The Field

A social coordination app that gets people into real-world experiences with strangers.

Not a matchmaker. A door.

## Status
Pre-development. Funding demo in progress.

## What this is
Quest-based event discovery with mutual matching and mandatory reasoning fields. 
Geography and time as primary filters. AI-assisted onboarding and post-experience debrief.

## What this isn't
A dating app. A networking tool. An optimization engine.
